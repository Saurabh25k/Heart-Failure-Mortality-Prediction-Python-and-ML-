# ❤️ Heart Failure Mortality Prediction using Machine Learning

A machine learning project that predicts the likelihood of mortality in heart failure patients using clinical data. Multiple classification algorithms were implemented and compared to determine the most effective model for prediction.



## 📌 Project Overview

Heart failure is one of the leading causes of death worldwide. Early prediction of patient survival can help healthcare professionals make informed treatment decisions and improve patient care.

This project uses machine learning techniques to analyze clinical patient data, preprocess the dataset, train multiple classification models, and evaluate their performance using standard metrics.



## 🎯 Objectives

- Predict the survival outcome of heart failure patients.
- Compare the performance of multiple machine learning algorithms.
- Evaluate each model using classification metrics.
- Identify the most accurate model for mortality prediction.



## 📂 Dataset

**Dataset:** `heart_failure_clinical_records_dataset.csv`

The dataset contains clinical records of heart failure patients with features such as:

- Age
- Anaemia
- Creatinine Phosphokinase (CPK)
- Diabetes
- Ejection Fraction
- High Blood Pressure
- Platelets
- Serum Creatinine
- Serum Sodium
- Sex
- Smoking
- Follow-up Time
- **DEATH_EVENT** (Target Variable)
